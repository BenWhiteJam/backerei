Tooling for the Cryptium Tezos Bäckerei

Coming soon!

### Development

[Stack](https://haskellstack.org) required.

#### Building

```bash
stack build
```

#### Testing

```bash
stack test
```

#### Installing

`~/.local/bin` will need to be on your `$PATH`.

```bash
stack install
```
